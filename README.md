# Петушкова Дарья ЭФБО-04-24

## Запуск проекта

Проект доступен онлайн: **https://fetrias.github.io/Color_generator/**

### Локальный запуск

**Требования:**
- Node.js 20.x или выше
- npm

**Установка зависимостей:**
```sh
npm install
```

**Основные зависимости:**
- Vue 3.5+
- Vue Router 4.x
- Vite 7.x

Если `vue-router` не установился автоматически, выполните:
```sh
npm install vue-router@4
```

**Запуск dev-сервера:**
```sh
npm run dev
```

**Сборка для production:**
```sh
npm run build
```
